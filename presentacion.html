JS – Recuperar modo oscuro

Se verifica si el usuario había activado el modo oscuro previamente.


if (localStorage.getItem('darkMode') === 'true') {
  body.classList.add('dark');
  modoToggle.checked = true;
  icon.textContent = '🌙';
} else {
  icon.textContent = '☀️';
}

JS – Alternar modo oscuro

Se cambia el modo de visualización y se guarda la preferencia del usuario.


modoToggle.addEventListener('change', () => {
  body.classList.toggle('dark');
  localStorage.setItem('darkMode',
    body.classList.contains('dark'));
  icon.textContent = body.classList.contains('dark') ? '🌙' : '☀️';
});

JS – Animación de secciones

Uso de IntersectionObserver para hacer visible cada sección al desplazarse.


const sections = document.querySelectorAll('section');
const observer = new IntersectionObserver(entries => {
  entries.forEach(entry => {
    if (entry.isIntersecting) entry.target.classList.add('visible');
  });
});
sections.forEach(sec => observer.observe(sec));
